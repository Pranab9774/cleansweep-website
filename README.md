# CleanSweep Website

Modern static website for CleanSweep.

## Deploy on GitHub Pages

1. Upload all files and folders to your `cleansweep-website` repository.
2. Go to **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose branch: `main`, folder: `/root`.
5. Save.
