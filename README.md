# CleanSweep Professional Website

Static, production-ready website for GitHub Pages.

## Upload instructions

Upload all files and the complete `assets` folder to the `main` branch of your GitHub repository.

Required structure:

```
index.html
privacy.html
terms.html
support.html
contact.html
delete-account.html
assets/
  styles.css
  main.js
  cleansweep-logo.png
  favicon.svg
```

Then go to Settings → Pages → Deploy from branch → main → /(root).
